## Tic-Tac-Toe Web Application 

This is an interactive and responsive Tic-Tac-Toe web application that allows users to play either Player vs Player or Player vs Computer.

## Features

- Player vs Player mode
- Player vs Computer mode (basic AI)
- Winning condition detection and highlight winning cells
- Restart / Reset game functionality

## Technologies Used

- HTML
- CSS Styling 
- JavaScript 

## How It Works

- Player selects a cell and a mark is placed (X or O).
- The game checks for winning combinations.
- If a player wins, the winning cells are highlighted.
- If all cells are filled, the game declares a draw.
- In Computer mode, the AI automatically selects an available cell.
